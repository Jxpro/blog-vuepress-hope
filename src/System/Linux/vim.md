# Vim基本使用

## 快速选择命令

| 命令 | 功能                             |
| ---- | -------------------------------- |
| w    | 移动到下一个单词头部             |
| e    | 移动到下一个单词尾部             |
| b    | 移动到前一个单词头部             |
| ge   | 移动到前一个单词尾部             |
| W    | 向后移动到以空格分隔的字符串头部 |
| E    | 向后移动到以空格分隔的字符串尾部 |
| B    | 向前移动到以空格分隔的字符串头部 |
| gE   | 向前移动到以空格分隔的字符串尾部 |
